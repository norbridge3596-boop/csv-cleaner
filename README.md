# csv-cleaner

Automated CSV cleaning and standardisation.

## Features
- Auto-detects delimiters (comma, tab, semicolon, pipe)
- Strips whitespace and inconsistent quoting
- Normalises dates to YYYY-MM-DD
- Handles missing values consistently
- Removes duplicate rows

## Usage
```bash
python3 csv_cleaner.py input.csv output.csv
```
